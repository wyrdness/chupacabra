All notable changes to the Chupacabra documentation.

### Added
- Initial api.json with comprehensive Chupacabra data
- README.md documentation
- SOURCES.md bibliography
- Repository structure with sightings directories
- GitHub issue and PR templates

### Data Coverage
- Phenomenon overview and etymology
- Two variant morphologies (Puerto Rican reptilian, Continental canine)
- Physical characteristics and abilities
- Geographic distribution across Americas
- Historical timeline from 1975 precursors to present
- Notable sightings including Tolentino and Cuero cases
- Physical evidence analysis (carcasses explained as mange)
- Scientific, paranormal, and skeptical theories
- Cultural impact and popular culture references
- Research community documentation
